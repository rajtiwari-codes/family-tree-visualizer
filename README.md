# Family Tree Manager

Interactive family tree web app to create, edit, visualize, and export genealogy data with a responsive UI and local data persistence.

## Tech Stack
- Frontend: HTML, CSS, JavaScript (Vanilla JS)
- Backend: None (client-side app)
- API: None
- Database: Browser localStorage

## Features
- Add, edit, and delete family members
- Parent-child tree visualization with connection lines
- Expand/collapse member cards
- Upload profile images
- Search members
- Zoom and pan canvas
- Multiple layout options
- Import/Export family data in JSON
- Basic family statistics

## Project Structure
- `family-tree.html` - Main UI page (recommended public version)
- `app.js` - Core app logic
- `1.html` - Older/similar variant
- `2.html` - Older/similar variant

## Getting Started
1. Clone/download the project.
2. Open `family-tree.html` in your browser.
3. Start building your family tree.

## Data Storage
- Data is saved in browser `localStorage`.
- Clearing browser storage will remove saved tree data.
- Use Export JSON for backup.
